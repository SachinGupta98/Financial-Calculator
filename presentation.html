07 — Security & Reliability

Production-Grade Security

FinFlap is built with security-first principles — no shortcuts taken.

🔐 Bcrypt Password Hashing

Passwords are never stored in plaintext. Industry-standard bcrypt algorithm with adaptive cost factor.

⏱️ Timed Reset Tokens

Password reset links generated with itsdangerous URLSafeTimedSerializer expire after 30 minutes automatically.

🛡️ Brute-Force Protection

Flask-Limiter enforces 10 login attempts per minute per IP. Exceeding the limit returns HTTP 429.

🔑 SECRET_KEY Enforcement

App raises a RuntimeError at startup if SECRET_KEY is missing — preventing accidental insecure deployments.

📤 File Upload Safety

All file uploads (OCR receipts) are capped at 5 MB via Flask's MAX_CONTENT_LENGTH to prevent DoS attacks.

👤 User Isolation

All API endpoints check current_user with @login_required. Users can only access their own data — no cross-account leakage.
