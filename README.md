# Financial-Calculator
FinFlap is a comprehensive, AI-powered personal finance management application designed primarily for the Indian market. It serves as a digital companion to help users track expenses, plan investments, and manage their overall financial health.

Key aspects of the application include:

Financial Tools: It features specialized calculators for EMI, SIP, Income Tax (India-specific), Retirement, and Compound Interest.

Asset Management: Users can manage their portfolios and set specific financial goals within the app.

Smart Features: The project incorporates an AI Financial Advisor and OCR (Optical Character Recognition) capabilities for scanning receipts or financial documents.

Modern Web Stack: Built using a Flask (Python) backend and a responsive Tailwind CSS frontend, it is also designed as a Progressive Web App (PWA) for mobile-friendly use.

Security & Privacy: The system includes secure user authentication, brute-force protection, and the ability to export financial data to Excel or CSV formats.
